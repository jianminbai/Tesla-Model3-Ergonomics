# 导航地图

本页面说明不同读者应该如何使用本项目。

## 我现在开车不舒服

优先看：

1. [症状快速排查表](quick_diagnosis_table.md)
2. [第七章 症状决策树](../book/07-症状决策树.md)
3. [第五章 座椅调节流程](../book/05-座椅调节流程.md)

目标：

```text
先判断机制，再决定调高度、前后、前沿、靠背还是腰托。
```
